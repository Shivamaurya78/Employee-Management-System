# Employee Management System

A **full-stack web application** for managing employee records, built using **Spring Boot** (backend) and **Angular** (frontend).

## Features

- Add, update, delete, and view employees
- Search and filter employees by name, department, or role
- Responsive frontend interface
- RESTful APIs for backend operations
- Easy to deploy and maintain

## Tech Stack

- **Backend:** Java, Spring Boot, Spring Data JPA, Maven
- **Frontend:** Angular
- **Database:** MySQL / H2 (configurable)
- **Version Control:** Git & GitHub
